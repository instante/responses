Requirements
------------

- PHP 5.4 or higher

- [Nette Framework](https://github.com/nette/nette) 2.2



Installation
------------

The best way to install Instante responses is using  [Composer](http://getcomposer.org/):

```sh
$ composer require instante/responses
```
